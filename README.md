# 💱 Telegram Kyat ↔ Baht Exchange Bot

A simple Telegram bot for converting Myanmar Kyat (MMK) and Thai Baht (THB).

Built with Node.js and node-telegram-bot-api.

---

## 🚀 Features

- 📊 Check current exchange rate
- 💱 Convert Kyat ➜ Baht
- 💱 Convert Baht ➜ Kyat
- 👑 Admin can update exchange rate
- 💾 Stores rate in data.json
- 🔐 Uses environment variables (.env)
